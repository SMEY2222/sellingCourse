# рҹҺ“ CoursEduVist - Final Project 

CoursEduVist бһӮбһәбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбһ¶бһҳбһўбһ“бһЎбһ¶бһү бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ (Final Project)бҹ” бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷ Admin бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбҹҒбһҡбҹҖбһ“ бһ“бһ·бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҹбһ·бһҹбҹ’бһҹбһ‘бһ·бһүбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲ Stripeбҹ”

---

##  бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (Tech Stack)

*   **Frontend:** React.js (Vite), Tailwind CSS, Framer Motion.
*   **Backend:** Node.js, Express.js.
*   **Database:** MongoDB Atlas (Cloud Database).
*   **Authentication:** JSON Web Token (JWT) & Cookies.
*   **Payment Integration:** Stripe API.

---

## вҡҷпёҸ бһҡбһ”бҹҖбһ”бһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Setup & Execution)

бһҹбһјбһҳбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҮбҹҶбһ бһ¶бһ“бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

### бҹЎ.  Database (Database Setup)
 **MongoDB Atlas**бҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бһүбҹ’бһ…бһјбһӣ SQL Script бһ‘бҹҒбҹ” 
*   бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ Database бһӣбһҫ MongoDB Atlas бһҡбһҪбһ…бһҷбһҖ Connection String бһҳбһҖбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„ file `.env` бһҖбҹ’бһ“бһ»бһ„ Folder backendбҹ”

### бҹў. Backend (Server)
```bash
cd backend
npm install
# бһ”бһ„бҹ’бһҖбһҫбһҸ file .env бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ folder backend бһҡбһҪбһ…бһҠбһ¶бһҖбҹӢбҹ–
# PORT=4001
# MONGO_URI=your_mongodb_uri
# JWT_SECRET=your_secret_key
# STRIPE_SECRET_KEY=your_stripe_secret_key
npm start
```
*Backend Server бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫ Port 4001*

### бҹЈ. бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ“бҹӮбһҖ Frontend (Client)
```bash
cd frontend
npm install
npm run dev
```
*Frontend бһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫ (http://localhost:5173)
